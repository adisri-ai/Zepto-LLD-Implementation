# Project Overview  
1. This project is an LLD Implmentation of Zepto app using LLD in Java.
2. The app has been divided into two components for LLD implementation :
    a. **ZEPTO** : The implementation of the main app including all the functionalities.
    b. **Payment Gateway** : Implementation of payments on placing of order
